# Ludusgame 🎮

Ludusgame est une plateforme de jeux en ligne interactive développée dans un cadre scolaire. Le projet propose une variété de jeux accessibles directement via le navigateur, avec un système de gestion d'utilisateurs.

## Développeur

Ce site a été entièrement conçu et développé par **Rahmouni Mohamed Khalil**.

## Technologies Utilisées

### Frontend
- **HTML5 & CSS3** : Structure et mise en page responsive.
- **JavaScript (Vanilla)** : Logique des jeux et interactions dynamiques.

### Backend
- **Node.js & Express** : Serveur et API.
- **MongoDB & Mongoose** : Base de données pour la gestion des utilisateurs et des scores.
- **CORS & Dotenv** : Sécurité et gestion des variables d'environnement.

## Fonctionnalités

- **Système d'Authentification** : Inscription et connexion sécurisées.
- **Bibliothèque de Jeux** : Plusieurs jeux interactifs disponibles (`Save The Recipe`, `Fragment`, `Asphalt`).
- **Interface Intuitive** : Design moderne et expérience utilisateur fluide.

## Installation

Pour lancer le projet localement :

### Backend
1. Naviguez dans le dossier `back` :
   ```bash
   cd back
   ```
2. Installez les dépendances :
   ```bash
   npm install
   ```
3. Lancez le serveur :
   ```bash
   npm run dev
   ```

### Frontend
Ouvrez simplement le fichier `front/index.html` dans votre navigateur ou utilisez une extension comme *Live Server*.
